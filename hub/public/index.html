🚀 Create Your Node How it Works

Get Your Node Running in 3 Steps

~2 min 1

Create Your Site

Open a terminal and run this command. Replace yourusername with your GitHub username: 

npx degit twilightscapes/piratesocial/node-template yourusername.github.io
cd yourusername.github.io
npm install
npm run setup

The setup wizard asks for your name, bio, and camera — then configures your site URL, CMS, hub connection, and deploy settings automatically.

💡 Need Node.js? Install v18+ first.
~1 min 2

Push to GitHub

Create a new repo on GitHub named yourusername.github.io, then push:

git init && git add -A && git commit -m "My pirate node"
git remote add origin https://github.com/yourusername/yourusername.github.io.git
git branch -M main && git push -u origin main
⚡ The repo must be named yourusername.github.io for GitHub Pages to work automatically.
~1 min 3

Enable GitHub Pages & Sign In

In your GitHub repo:

Settings → Pages → Source: select "GitHub Actions"

Wait for the first deploy (check the Actions tab). Once live, visit https://yourusername.github.io and click "Sign In" to join the network.

✍️ Manage your posts at yourusername.github.io/admin — no coding needed.

How Pirate Social Works

🏠

Your Own Site

Each member hosts their own photography site on GitHub Pages — free, fast, and fully yours.

📡

RSS Federation

Sites publish RSS feeds. The hub aggregates them into a shared timeline — no centralized database for your content.

🤝

Social Hub

Likes, comments, and follows are coordinated through this hub. Your posts stay on your site.

🦋

Bluesky Bridge

Optionally cross-post to Bluesky. Comments from both networks appear together on your site.

✍️

Built-in CMS

Write posts with Decap CMS — a visual editor right in your browser, backed by git.

📷

EXIF & Galleries

Automatic camera data extraction, photo galleries, and slideshows built in.
